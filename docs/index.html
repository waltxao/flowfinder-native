原生体验

核心功能

为 macOS 用户打造的全方位文件管理体验

🪟

玻璃态双栏

NSVisualEffectView 系统级毛玻璃材质，左右独立导航，自动跟随深浅色主题，完全匹配 macOS 原生视觉。

⚡

Rust Core 引擎

getattrlistbulk 批量读取、BLAKE3 哈希、rayon 并行处理，性能远超 WebView 方案，目录读取快 10-30 倍。

🔀

跨面板操作

一键复制（⌘⇧C）/ 移动（⌘⇧X）到对侧面板，文件夹可直接在对侧打开，冲突自动追加「副本 N」。

👁

Quick Look 预览

QLPreviewPanel 原生浮动预览，空格键即可快速查看文件内容，无需打开第三方应用。

🔍

重复文件检测

三阶段 BLAKE3 哈希检测（大小分组 → 部分哈希 → 完整哈希），实时进度，安全删除到废纸篓。

🤖

AI 智能打标

支持 OpenAI / Claude / Ollama 多模型，隐私隔离仅发送文件名，标签与 macOS 原生标签双向同步。

📁

原生拖拽

NSDraggingSource/Destination，同卷移动、跨卷复制、Cmd 键切换，行为与 Finder 完全一致。

💾

APFS CoW 复制

利用 clonefile 系统调用实现 APFS 卷零拷贝复制，大文件瞬时复制完成，不占用额外磁盘空间。

🌐

SMB 网络共享

渐进式加载、LRU 目录缓存、rayon 并行操作、断连自动重连，流畅访问网络驱动器。

架构设计

Swift & AppKit 负责 UI，Rust Core 提供高性能后端，FFI 桥接
